Grupos de botões Botões em grupos para funcionalidades parecidas com barra de ferramentas

Grupos de botões

Use grupos de botões para unir vários botões em um elemento composto. Eles podem ser construídos com uma série de tags <a> ou elementos <button>.

Melhores práticas

Nós recomendamos os seguintes guidelines para usar grupos de botões e barra de ferramentas

  • Sempre use algum elemento em um único grupo de botões, pode ser <a> ou <button>.
  • Não misture botões de cores diferentes no mesmo grupo de botões
  • Use ícones para complementar ao invés de textos, mas inclua o atributo alt e title quando apropriado

Relacionado Grupos de botões com dropdown (veja abaixo) deve ser chamado separadamente e sempre inclua um dropdown para indicar uma intenção desejada.

Exemplo padrão

Aqui está como o HTML se parece como botão padrão para construir um grupo de botões com a tag button

<div class="btn-group">
  <button class="btn">1</button>
  <button class="btn">2</button>
  <button class="btn">3</button>
</div>

Exemplo de barra de ferramentas

Combine as escolhas da estrutura <div class="btn-group"> em um <div class="btn-toolbar"> para componentes mais complexos.

<div class="btn-toolbar">
  <div class="btn-group">
    ...
  </div>
</div>

Padrões leves Mesmo markup, classes diferentes

Classe base poderosa

All nav components here—tabs, pills, and lists—share the same base markup and styles through the .nav class.

Quando usar

Tabs and pills are great for sections of content or navigating between pages of related content.

Alinhamento do componente

To align nav links, use the .pull-left or .pull-right utility classes. Both classes will add a CSS float in the specified direction.

Barra de navegação fixa

Fix the navbar to the top or bottom of the viewport with an additional class on the outermost div, .navbar.

<div class="navbar navbar-fixed-top">
  ...
</div>

<div class="navbar navbar-fixed-bottom">

When you affix the navbar, remember to account for the hidden area underneath. Add 40px or more of apdding to the <body>. Be sure to add this after the core Bootstrap CSS and before the optional responsive CSS.

Nome da marca

Um link simples para mostrar sua marca ou nome do projeto requer apenas uma tag de link

<a class="brand" href="#">
  Nome do projeto
</a>

Formulários na barra de navegação

Para estilizar e posicionar apropriadamente um formulário dentro da barra de navegação, adicione a classe apropriada como mostrado abaixo. Para um formulário padrão, inclua .navbar-form ou .pull-left ou .pull-right para alinhar

<form class="navbar-form pull-left">
  <input type="text" class="span2">
</form>

Para um formulário de busca mais customizado, adicione a classe .navbar-search para receber estilos especializados na barra de navegação

<form class="navbar-search pull-left">
  <input type="text" class="search-query" placeholder="Busca">
</form>

Variação responsiva opcional

Dependendo da quantidade de conteúdo na sua barra do topo, você pode querer implementar uma opção responsiva. Para fazer isto, coloque em volta do conteúdo uma div, .nav-collapse.collapse, e adicione o botão alternado da barra de navegação, .btn-navbar.

<div class="navbar">
  <div class="navbar-inner">
    <div class="container">

      <!-- .btn-navbar é usado como alternador para conteúdo de barra de navegação colapsável -->
      <a class="btn btn-navbar" data-toggle="collapse" data-target=".nav-collapse">
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
      </a>

      <!-- Tenha certeza de deixar a marca se você quer que ela seja mostrada -->
      <a class="brand" href="#">Nome do projeto</a>

      <!-- Tudo que você queira escondido em 940px ou menos, coloque aqui -->
      <div class="nav-collapse">
        <!-- .nav, .navbar-search, .navbar-form, etc -->
      </div>

    </div>
  </div>
</div>
Atenção! The responsive navbar requires the collapse plugin.

Breadcrumb navigation is used as a way to show users where they are within an app or a site, but not for primary navigation. Keep their use sparse and succinct to be most effective.

Paginação Duas opções para paginar pelo conteúdo

Paginação de página multi-ícones

Quando usar

Ultra simplistic and minimally styled pagination inspired by Rdio, great for apps and search results. The large block is hard to miss, easily scalable, and provides large click areas.

Links de página sem estado

Links are customizable and work in a number of circumstances with the right class. .disabled for unclickable links and .active for current page.

Alinhamento flexível

Add either of two optional classes to change the alignment of pagination links: .pagination-centered and .pagination-right.

Por que usar miniaturas

As miniaturas (previamente .media-grid até a versão 1.4) são ótimos para grids de fotos ou vídeos, imagens em resultados de busca, detalhes de produtos, portfolios e muito mais. Eles podem ser links ou conteúdo estático

marcação simples e flexível

A marcação da miniatura é simplesmente uma ul com qualquer número de elementos li. É também super flexível, permitindo para qualquer tipo de conteúdo com um pouco mais de markup para colocar no seu conteúdo

use tamanho das colunas do grid

Por último, os componentes de miniatura usam o grid já existente —como .span2 ou .span3— para controlar as dimensões da miniatura

A marcação

Como mencionado antes, a marcação necessária para as miniaturas é leve e direta. Dê uma olhada na configuração padrão para imagens linkadas

<ul class="thumbnails">
  <li class="span3">
    <a href="#" class="thumbnail">
      <img src="http://placehold.it/260x180" alt="">
    </a>
  </li>
  ...
</ul>

Para conteúdo personalizado nas miniaturas, a marcação muda um pouco. Para permitir elementos bloco em qualquer lugar, nós trocamos a tag <a> para uma div <div> assim:

<ul class="thumbnails">
  <li class="span3">
    <div class="thumbnail">
      <img src="http://placehold.it/260x180" alt="">
      <h5>Rótulo para a miniatura</h5>
      <p>A descrição da miniatura vem aqui...</p>
    </div>
  </li>
  ...
</ul>

Alertas Estilos para sucesso, advertência, e mensagens de erro

Padrões leves

Reescrita a classe base

Com o Bootstrap 2, nós simplificamos a classe base: .alert ao invés de .alert-message. Nós também reduzimos o mínimo requerido de marcação na tag <p> que é requerido por padrão, simplesmente coloque em torno uma <div>.

Alerta de mensagem único

Para um componente mais durável e com menos código, nós removemos o visual diferenciado para blocos de alertas, mensagens que vinham com mas padding e tipicamente mais texto. A classe também mudou a classe .alert-block.

Cai bem com javascript

O Bootstrap vem com um plugin ótimo de jQuery que suporta mensagens de alerta, dispensando-os de uma forma simples e fácil

Barra de progresso Para carregamento, redirecionamento, ou ação de status

Exemplos e marcação

Básico

Barra de progresso padrão com um gradiente vertical

<div class="progress">
  <div class="bar"
       style="width: 60%;"></div>
</div>

Zebrada

Use um gradiente para criar um efeito de zebra (nada de IE)

<div class="progress progress-striped">
  <div class="bar"
       style="width: 20%;"></div>
</div>

Animado

Pegue um exemplo zebrado e animeo (nada de IE)

<div class="progress progress-striped
     active">
  <div class="bar"
       style="width: 40%;"></div>
</div>

Opções e suporte a browser

Cores adicionais

Barras de progresso use algumas das classes de botões e alertas para estilos consistentes

Barras zebradas

Similar as cores sólidas, nós zebramos as barras de progresso

Comportamento

Barra de progresso usam transições CSS3, então você ajusta dinamicamente a largura via javascript, que ele irá suavemente ser redimensionado.

Se você usa a classe .active, sua barra de progresso irá animar zebrada da esquerda pra direita com a classe .progress-striped.

Suporte a browser

Barra de progresso usa gradientes CSS3, transitions, e as animações são obtidas com todos os seus efeitos. Estas funcionalidades não são suportadas no IE7-9 ou versões mais antigas de Firefox.

Opera e IE não suporta animações desta forma
